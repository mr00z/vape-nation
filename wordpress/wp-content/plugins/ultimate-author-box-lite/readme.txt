== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ultimate-author-box-lite` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Dashboard->Ultimate Author Box Lite screen to configure the plugin.
4. Go to Dashboard-> Users-> Profile to set up your user profile.
